# ShadowLayout

## Dependency

Step 1. Add the JitPack repository to your build file:
```groovy
    allprojects {
        epositories {
            maven { url 'https://jitpack.io' }
        }
    }
```

Step 2. Add the dependency:
```aidl
    dependencies {
            compile 'com.github.fbsum:shadowlayout:1.0.0'
    }
```
